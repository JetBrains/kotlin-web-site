In this resource center you can find documentation, tutorials, videos and more information about Kotlin

  • Reference. Syntax reference and information on basic constructs with Kotlin
  • Tutorials. Walk-through tutorials providing step-by-step guidance on how to accomplish different tasks
  • Videos. Short (and longer) videos showing different aspects of Kotlin
  • Events. Upcoming events where presentations on Kotlin are given, as well as past events with links to videos and slides
  • More Resources. Links and information to some community resources
