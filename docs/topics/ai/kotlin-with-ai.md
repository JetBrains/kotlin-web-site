You can use one approach, or both together in the same project.
This page explains each option and helps you decide which to use.

## Build AI-powered apps with Kotlin

Use Kotlin to create applications that use AI capabilities.

The Kotlin ecosystem provides libraries, frameworks, and integrations for working
with AI services and models across different platforms.

To learn how to build AI-powered applications with Kotlin, see:

* [](kotlin-ai-apps-development-overview.md)
* [](spring-ai-guide.md)

## Develop with AI tools

AI-powered development tools can help you write, test, and maintain your Kotlin code.

Kotlin supports AI-assisted development through IDE integrations, reusable AI skills,
and third-party AI tools like Claude Code, OpenAI Codex, and more.

See [AI-powered Kotlin development](ai-for-development.md) for an overview of these tools and how to use them.

### Kotlin AI skills

Kotlin AI skills are reusable, task-specific instructions that help AI agents perform Kotlin development tasks consistently.

You can provide AI skills to compatible AI agents to:

* Explain Kotlin code.
* Write tests.
* Migrate APIs.
* Apply Kotlin best practices.

For more information, see [](kotlin-ai-skills.md).
